1. html: The root element of an HTML document. Contains all other HTML elements on the page.
  2. title: Defines the title of the HTML document, displayed in the browser's title bar or tab. Important for SEO and bookmarking.
  3. body: Contains the visible content of the web page, including text, images, links, and more.
  4. p: Represents a paragraph of text.
  5. a: Defines a hyperlink. Allows users to navigate to another web page, resource, or location within the same page.
